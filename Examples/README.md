# Examples

Canonical persona pack:
- `Examples/personapad.pack.json`

Canonical compose command:
```bash
swift run personapad compose --persona senior-ios-engineer --context "Repo: PersonaPad" --goal "Ship v1" --task "Review changes"
```

To load this pack in the app:
1) Copy `Examples/personapad.pack.json` to `~/Library/Application Support/PersonaPad/Packs/`
2) Click **Reload** in the app toolbar
