# Feature Brief Template

Use this template for `venture-product-discovery` outputs.

## Required Sections

1. Problem Statement
2. Target User And Context
3. Desired Outcome
4. Proposed Scope (in/out)
5. Risks And Unknowns
6. Dependencies
7. Acceptance Criteria
8. Decision Gate
9. First Next Action

## Minimum Quality Bar

1. Problem statement is falsifiable and user-centered.
2. Scope includes explicit out-of-scope boundaries.
3. Risks include at least one mitigation or data need.
4. Acceptance criteria are observable and testable.
5. Decision gate names the approver and approval condition.
