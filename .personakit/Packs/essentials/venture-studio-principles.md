# Venture Studio Principles

Use this essential when evaluating and selecting business opportunities.

## Objectives

1. Generate high-quality opportunities from real user pain.
2. Prioritize with transparent, repeatable scoring.
3. Select one focus opportunity at a time.
4. Move from idea to MVP scope quickly and ethically.

## Operating Rules

- Work in plain language first.
- Prefer concrete evidence over assumptions.
- Keep scope bounded to the active MVP cycle.
- Record rationale for every prioritize/park/drop decision.

## Ethics Gate

Before selecting any opportunity, verify:

1. It respects user dignity and privacy.
2. It does not rely on deceptive patterns.
3. It is defensible under a simple public explanation.

## Output Requirements

Every cycle must produce:

- three opportunity candidates
- one scorecard table
- one selected candidate (or explicit no-go)
- one 14-day MVP outline
