Status: Active

## Purpose

`Workspaces/` isolates initiative subprojects from the core PersonaKit product
root. Each workspace is expected to manage its own documentation and
`.personakit` root.

## Current Workspaces

- [VentureStudio](./VentureStudio/): initiative and storytelling workspace.

## Workspace Rules

1. Keep workspace-specific packs and sessions under that workspace's `.personakit`.
2. Keep initiative docs under that workspace's `Docs/`.
3. Avoid mixing workspace-only artifacts back into root `Docs/` unless promoted.
