# ReleaseDesk PersonaKit Root

This public root is the completed operating-contract set from the ReleaseDesk
tutorial. It is published as `personakit-root` so static hosts display it.

In a real project, the same contents would usually live in `.personakit/`.

The `releasedesk-readiness` skill declaration points at a host-local skill in
the sample app. PersonaKit declares whether that skill is allowed for a session;
the host remains responsible for using it.
