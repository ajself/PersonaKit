Built on Three Editorial Engineering Rules

1. Identity First

Persona and session context are explicit. No identity drift. No hidden assumptions.

2. Scope Must Be Deterministic

Local-first resolution ensures the active project context stays in front unless an explicit override says otherwise.

3. Review Is the Engine

Agent velocity is useful only when paired with disciplined review, tight diffs, and measurable verification.
