## Purpose

Retain completed or superseded planning artifacts for historical reference
within the PersonaKit archive.

## Rule of Use

1. Move a plan here once it no longer drives the next decision.
2. Keep statuses honest (`Archived`, `Complete`, or similar historical state).
3. Active work should point here only for baseline context, not for current
   execution control.
