# Orbit Retrospective Packets

Status: Draft
Owner: Samwise

## Purpose

Provide runnable templates for Orbit retrospective work.

These templates support:

- shared evidence packet creation
- roundtable retrospective execution
- fan-out retrospective execution
- head-to-head scoring and comparison

## Usage Order

1. Create the shared evidence packet.
2. Run either the roundtable or fan-out method.
3. Record the scorecard.
4. Write the comparison decision note.
