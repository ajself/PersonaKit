   calling any Orbit surface "command center" quality.
4. Start measuring checkpoint completeness against the four-surface spec, not
   just what was easy to ship.

   bounded single-agent delivery inside a multiagent-ready framework.
2. Stop treating snapshot-backed UI and deterministic tests as sufficient
   evidence of product quality.

   emphasis to compensate for unclear interaction design.
4. Stop treating a retrospective document as evidence that the retrospective
   loop itself ran.
