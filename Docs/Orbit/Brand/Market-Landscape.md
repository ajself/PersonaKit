# Market Landscape

Orbit enters an emerging space at the intersection of several existing AI software categories. Understanding these adjacent markets helps clarify where Orbit fits — and where it deliberately diverges.

The key insight is that most current products treat AI as a **tool** or **automation layer**, while Orbit treats AI as **collaborators organized into teams**.

---

# Current Market Categories

## 1. AI Chat Assistants

Examples:

- ChatGPT
- Claude
- Gemini

These tools provide powerful conversational interfaces for interacting with large language models.

Strengths:

- easy to use
- flexible general-purpose interaction
- strong reasoning capabilities

Limitations:

- no persistent roles
- no structured teams
- no durable collaboration model
- limited long‑term memory across projects

These tools are best understood as **AI conversation engines**, not collaboration systems.

---

## 2. Agent Frameworks

Examples:

- LangGraph / LangChain
- CrewAI
- Microsoft AutoGen

These frameworks allow developers to build systems where multiple AI agents interact with each other.

Strengths:

- flexible orchestration
- programmable workflows
- strong developer control

Limitations:

- infrastructure focused
- no user-facing collaboration model
- no workspace concept
- little support for persistent identity or memory governance

These systems are best described as **developer infrastructure for agents**, not tools for working with AI teams directly.

---

## 3. AI‑Augmented Collaboration Tools

Examples:

- Slack AI
- Notion AI
- Linear AI

These tools integrate AI features into existing collaboration platforms.

Strengths:

- familiar workflows
- integration with documents and messaging

Limitations:

- AI is an assistant, not a participant
- no persistent AI identities
- no multi-agent collaboration model
- no structured memory evolution

These products treat AI as a **feature inside existing software**, rather than the core participant in collaboration.

---

# The Gap in the Market

The existing landscape reveals a clear gap.

Current tools offer:

- conversations with AI
- frameworks for building agents
- collaboration tools with AI assistance

But very few systems provide:

- persistent AI teammates
- structured AI teams
- meeting-style collaboration
- durable shared memory
- workspace‑scoped learning

This gap creates the opportunity for a new category.

---

# Orbit's Position

Orbit focuses on a different concept:

**AI teams with persistent identity, memory, and coordination.**

Instead of interacting with isolated AI assistants, users interact with **teams of specialized AI collaborators**.

Key characteristics:

- personas represent persistent roles
- teams and squads organize collaboration
- meetings coordinate discussion
- journals capture reflection
- memory evolves through review and promotion

Orbit therefore shifts the mental model from:

AI as tools → AI as teammates

---

# Category Definition

Orbit defines a new category that can be described as:

**AI Team Operating System**

An AI Team Operating System provides:

- persistent AI roles
- workspace organization
- coordinated multi‑agent collaboration
- durable memory
- human‑directed governance

In this model, the human remains the center of gravity, while AI collaborators operate as members of a structured team.

---

# Strategic Differentiation

Orbit differentiates itself through several principles:

Human‑Centered Control

The user remains the authority over decisions, memory, and team structure.

Persistent Identity

AI collaborators maintain roles, history, and expertise over time.

Structured Collaboration

Work happens through meetings, squads, and teams rather than isolated prompts.

Durable Knowledge

Learning evolves through journals and reviewed memory rather than uncontrolled accumulation.

---

# Summary

Orbit does not attempt to replace chat assistants, agent frameworks, or collaboration tools.

Instead, it sits above them as an environment where AI collaborators can operate together.

In short:

Chat tools enable conversations.

Agent frameworks enable automation.

Collaboration tools enable communication.

**Orbit enables AI teams.**
