# PersonaKit Docs

Status: Active
Owner: AJ
Last Reviewed: 2026-03-09

## Purpose

Provide the documentation map for PersonaKit as the engine and repository
operating model inside this codebase.

PersonaKit is not the forward product surface.
It is the engine, context system, and implementation/runtime core that Orbit
builds on.

## Contents

- [Architecture](./Architecture/README.md)
  Durable technical decisions, invariants, and domain maps for the PersonaKit
  repository and runtime.
- [Development](./Development/README.md)
  Workflow rules, logs, session routing, validation guidance, and collaboration
  records.

## Relationship To Orbit

- `Docs/Orbit/` is the current forward product direction.
- `Docs/PersonaKit/` describes the engine, implementation, and operating model
  that support that direction.
- Earlier PersonaKit planning and research artifacts live in
  `Docs/Archive/PersonaKit/`.
