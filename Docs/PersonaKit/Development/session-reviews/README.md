# Session Reviews

Status: Active  
Owner: AJ  
Last Reviewed: 2026-03-10

## Purpose

Store durable Samwise session-stack review artifacts for PersonaKit workflows.

These reviews are findings-first and should capture:

1. target session and normalized session id
2. ordered findings
3. SWOT per reviewed artifact
4. fixed risk-dimension scores
5. current and projected confidence
6. explicit verdict
7. red-pen recommendations

## Default Naming

- `YYYY-MM-DD-<normalized-session-id>.md`
