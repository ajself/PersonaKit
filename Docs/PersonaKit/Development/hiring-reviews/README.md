# Persona Hiring Reviews

Status: Active  
Owner: AJ  
Last Reviewed: 2026-03-08

## Purpose

Store human-readable reverse-interview reports for persona candidate assessments.

## Naming

Use:

- `YYYY-MM-DD-<personaId>.md`

Example:

- `2026-03-07-venture-product-steward.md`

## Required Sections

1. Role context and required capabilities.
2. Evidence references.
3. Strengths.
4. Gaps by severity.
5. Unknowns and assumptions.
6. Confidence score and threshold.
7. Verdict.
8. Missing artifact recommendations by type.
9. First implementation step.
